# AI Resources

A personal collection of resources related to AI, LLMs, and agents — curated and created by [@soenderby](https://github.com/soenderby). This repo serves as a central hub for sharing and syncing across machines.

## Structure

```
ai-resources/
├── papers/         # Research papers and summaries
├── prompts/        # Prompt templates and system prompts
├── tools/          # Scripts, tools, and utilities
├── agents/         # Agent designs, configs, and experiments
├── notes/          # Personal notes, write-ups, and observations
└── links/          # Curated link lists by topic
```

## Topics Covered

- Large Language Models (LLMs)
- AI Agents & multi-agent systems
- Prompt engineering
- Retrieval-Augmented Generation (RAG)
- Fine-tuning & training
- Tooling & infrastructure
- Research & papers
