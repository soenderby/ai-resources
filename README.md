# AI Resources

A personal collection of resources related to AI, LLMs, and agents.
It is a very interesting and fast developing area. I keep finding new interesting resources, and it got to the point where I had trouble keeping track of it. So I created this repository to gather and organize everything. By having everything in a single, version controlled, place, also means that I can have LLM agents help me maintain it.

## Structure

```
ai-resources/
├── librarian-notes.md     ← hot memory for startup orientation
├── librarian-archive.md   ← archived memory, read selectively
├── agent-skills/          ← reusable skills for pi and Claude Code
├── ai-generated/
│   ├── README.md          ← compact manifest for generated docs
│   └── ...                ← generated syntheses and analyses
└── resources/
    ├── index.md           ← compact inventory of tracked sources
    ├── articles.md        ← specific articles, posts, and reference sites
    ├── books.md           ← book-length sources worth keeping
    ├── people.md          ← people worth following
    └── projects.md        ← open source projects and tools
```
