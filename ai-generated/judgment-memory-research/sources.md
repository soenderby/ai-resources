## 1. Agent READMEs: An Empirical Study of Context Files for Agentic Coding
- **URL:** https://arxiv.org/html/2511.12884v1
- **Source type:** Academic paper (large-scale empirical study)
- **Term(s) used:** agent context files, agent READMEs, persistent long-term memory for the agent, agentic coding manifests

- **Relevance:** The single richest primary source found. Reports maintenance cost (~1-2 hrs/week) and failure modes (specification staleness causing silent failures). Includes context drift detector.
- **Status:** accepted
- **Reason:** Strongest inspectable artifact in the research. Covers production, storage, application, maintenance, and failure modes. The three-tier architecture is a concrete embodiment of judgment memory scaling.
