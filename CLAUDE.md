# Agent Instructions

This repository is a personal knowledge base for AI, LLMs, and agent-related resources. It is maintained by @soenderby and shared across machines and with others.

## Purpose

- Collect, organize, and summarize resources related to AI and LLMs
- Store reusable agent skills, prompts, and tooling
- Serve as a reference for agent and LLM development work

## How to Help

When working in this repo you should:

- **Prefer simplicity and clarity** — content here is meant to be readable and shareable
- **Use Markdown** for all documentation and notes
- **Keep files focused** — one topic or skill per file where possible
- **Suggest structure organically** — only propose new directories when there is actual content that warrants it; don't create empty scaffolding
- **Summarize external resources** — when adding links or papers, include a brief note on why it is relevant
- **agent-skills/** contains reusable skills for AI coding agents (e.g. pi, Claude Code). Each skill lives in its own subdirectory with a `SKILL.md` describing its purpose and usage

## Conventions

- Branch: `main`
- Commit messages: short and descriptive, present tense (e.g. `Add RAG overview note`)
- File names: lowercase, hyphen-separated (e.g. `rag-overview.md`)
